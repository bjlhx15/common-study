# java-serializer
# 序列化项目
## serialize-json-lib：json-lib框架
*    支持java-xml、java-json、xml-java、json-java
## serialize-jackson：Jackson框架
*    支持java-xml、java-json、xml-java、json-java
## serialize-xstream：XStream框架
*    支持java-xml、java-json、xml-java、json-java
## serialize-jaxb2：jaxb2
*    支持java-xml、xml-java  
    jdk6开始支持
